How to add questions and answers to the Chatbot

Where to add the questions and answers?

After purchasing the chatbot you will be given an intents.json file where you can add/edit questions and answers.

How does the intents.json file work?

After opening the file it should look something like this:

intents.json

{
    "intents": [
        {
            "tag": "timing",
            "patterns": [
                "What time are you open?",
            ],
            "responses": [
                "We are open 24/7",
            ],
            "context_set": ""
        }
    ]
}

All the intents are inside the intents list ("intents: []") and each of the are dictionaries contains the following values:

  • "tag" - The tag is like what the question is about. In the above example you can see the tag is "timing" because the question is about the timing
  • "patterns" - The patterns is a list of questions the user may ask to trigger this intent to be used
  • "responses" - The responses is a list of answers that will be shown. If there is more than 1 response in the "responses" list than any one will be choose randomly
  • "context_set" - The context set is necessary for the chatbot when being trained. It can be blank

How to add the questions and answer?

  1. First you will have to open up the intents.json file

  2. After opening the file it should look something like this:

    intents.json

    {
    "intents": [
        {
            "tag": "timing",
            "patterns": [
                "What time are you open?",
            ],
            "responses": [
                "We are open 24/7",
            ],
            "context_set": ""
        }
    ]
}

  3. Add another dictionary to the "intents" list

  4. After adding a new dictionary it will look something like this:

    intents.json

    {
    "intents": [
        {
            "tag": "timing",
            "patterns": [
                "What time are you open?",
            ],
            "responses": [
                "We are open 24/7",
            ],
            "context_set": ""
        },
        {
	        "tag": "location",
	        "patterns": [
		        "Where are you located?",
	        ],
	        "responses": [
		        "We are located in Tokyo, Japan",
	        ],
	        "context_set": ""
	    }
    ]
}

  5. Save the file

How to edit questions and answers?

  1. First you will have to open up the intents.json file

  2. After opening the file it should look something like this:

    intents.json

    {
    "intents": [
        {
            "tag": "timing",
            "patterns": [
                "What time are you open?",
            ],
            "responses": [
                "We are open 24/7",
            ],
            "context_set": ""
        }
    ]
}

  3. Change the "patterns" or "responses" (in this case we would like to change "We are open 24/7")

  4. After changing "We are open 24/7" to "We are open Monday to Friday 8AM - 10PM" it will look something like this:

    intents.json

    {
    "intents": [
        {
            "tag": "timing",
            "patterns": [
                "What time are you open?",
            ],
            "responses": [
                "We are open Monday to Friday 8AM - 10PM",
            ],
            "context_set": ""
        }
    ]
}

  5. Save the file

What features does the chatbot include?

Features that the chatbot include are:

  • The Chatbot is an API

  • It can be integrated with the website (sample at https://chatbot-api-fiverr-sample.herokuapp.com/)

  • It request other API’s

  • The questions and answers are easily customizable
